# Webpage Screenshot Tool

一个用 Python 编写的自动化网页截图工具，适用于开源项目、文档存档等场景。

## 功能
- 输入网址后自动截取全屏截图
- 无需手动操作，适合批量处理

## 如何使用
1. 安装依赖：
```bash
pip install -r requirements.txt
```
2.运行程序：
```
python screenshot.py
```
3.截图会保存在 screenshots/ 目录下

支持更多浏览器（计划中）……
